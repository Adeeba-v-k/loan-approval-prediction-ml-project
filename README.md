# Loan Approval Prediction

This project predicts whether a loan will be approved or not using applicant financial and demographic data.

## Dataset
- Kaggle Loan Prediction Dataset

## Tools & Technologies
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Jupyter Notebook  

## Model
- Random Forest Classifier

## Steps
- Data cleaning and preprocessing
- Label encoding of categorical variables
- Handling class imbalance using SMOTE
- Model training and evaluation using Random Forest

## Output
The model predicts whether a loan application will be **approved or not approved**.

## Result
- Achieved ~78% accuracy
- Credit History and Income were key influencing features
